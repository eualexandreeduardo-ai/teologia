# Como publicar este site

## Opção mais simples: GitHub + Vercel

1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta para o repositório.
3. Confirme que o arquivo `index.html` está na raiz do repositório.
4. Entre na Vercel.
5. Clique em **Add New Project**.
6. Escolha o repositório.
7. Framework: **Other**.
8. Build Command: deixe vazio.
9. Output Directory: deixe vazio ou coloque `.`.
10. Clique em **Deploy**.

Pronto. A Vercel vai gerar um link público para sua plataforma.
